# Calculadora Web Interactiva

## Características

- ✅ Operaciones básicas: suma, resta, multiplicación, división
- ✅ Operaciones avanzadas: porcentajes, raíz cuadrada, potencias
- ✅ Historial de operaciones
- ✅ Soporte para teclado físico
- ✅ Diseño responsive
- ✅ Manejo de errores (división por cero, etc.)
- ✅ Modo oscuro/claro
- ✅ Animaciones suaves

## Tecnologías

- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript ES6+

## Instalación

```bash
git clone https://github.com/tu-usuario/calculadora-web.git
cd calculadora-web
# Abrir index.html en tu navegador
```

## Uso

1. Haz clic en los botones numéricos o usa el teclado físico
2. Selecciona una operación (+, -, ×, ÷)
3. Presiona "=" o Enter para ver el resultado
4. Usa "C" para limpiar, "AC" para resetear completamente
5. Consulta el historial en el panel lateral
